# Resume Folder

This folder contains your downloadable resume file.

## Adding Your Resume

1. **Export your resume as PDF**
   - Use your preferred format (Word, Google Docs, LaTeX, etc.)
   - Export/Save as PDF format
   - Name it: `Vasanth_Panuganti_Resume.pdf`

2. **Add to this folder**
   - Copy your PDF file to this `resume/` folder
   - [Vasanth_Panuganti_Resume.pdf](https://github.com/user-attachments/files/23343511/Vasanth_Panuganti_Resume.pdf)
   - Replace the placeholder file if it exists

3. **The download button will automatically work**
   - The website is already configured to link to `resume/Vasanth_Panuganti_Resume.pdf`
   - Users can download it by clicking the "Download Resume" button

## File Naming

**Recommended filename**: `Vasanth_Panuganti_Resume.pdf`

If you prefer a different filename, update the link in `index.html`:
- Find: `href="resume/Vasanth_Panuganti_Resume.pdf"`
- Change to: `href="resume/YOUR_FILENAME.pdf"`

## Tips

- Keep file size under 2MB for faster downloads
- Use a professional, clean resume format
- Update the PDF whenever you update your resume
- Consider versioning (e.g., `Resume_2025.pdf`) if you maintain multiple versions
