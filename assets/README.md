# Assets Folder

This folder contains images and other assets for the portfolio website.

## Required Images

Please add the following images to this folder:

### 1. Profile Picture
- **Filename**: `profile.jpg`!
- ![Profile](https://github.com/user-attachments/assets/00500046-0be7-4abe-b219-5102be2d688c)

- **Recommended size**: 400x400 pixels (square)
- **Format**: JPG or PNG
- **Description**: Your professional headshot or portrait

### 2. Project Images
- **Filenames**: `project1.jpg`, `project2.jpg`, `project3.jpg`
- **Recommended size**: 600x400 pixels (3:2 aspect ratio)
- **Format**: JPG or PNG
- **Description**: Screenshots or preview images of your projects!


### 3. Case Study Images
- **Filenames**: `case-study-1.jpg`, `case-study-2.jpg`, `case-study-3.jpg`
- **Recommended size**: 600x400 pixels (3:2 aspect ratio)
- **Format**: JPG or PNG
- **Description**: Featured images for your detailed case studies

### 4. Company Logos (Work History)
- **Filenames**: `k12-logo.png`, `flipkart-logo.png`, `byjus-logo.png`
- **Recommended size**: 200x200 pixels (square)
- **Format**: PNG (with transparent background preferred)
- **Description**: Company logos for work history section
- **Note**: Logos should be official company logos

## How to Add Images

1. Copy your images to this folder
2. Rename them according to the filenames listed above
3. Or update the `src` attributes in `index.html` to match your filenames

## Image Optimization Tips

- Compress images before adding them to reduce file size
- Use JPEG for photographs and PNG for graphics with transparency
- Recommended tools: TinyPNG, ImageOptim, or Squoosh
- Keep file sizes under 500KB for optimal loading speed

## Fallback Behavior

If images are not found, the website will display:
- Profile picture placeholder with "Your Photo Here" text
- Project cards with gradient backgrounds
