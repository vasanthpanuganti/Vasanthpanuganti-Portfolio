# Vasanth Panuganti - Portfolio Website

Welcome to my professional portfolio website! This is a modern, responsive portfolio showcasing my skills, projects, and professional experience.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Easy Navigation**: Quick access toggles to jump to Resume and Career Highlights
- **Comprehensive Sections**:
  - Hero section with headline and profile picture
  - Career Highlights
  - Detailed Resume
  - Project Showcase
  - Case Studies (detailed project breakdowns)
  - Certifications
  - Extra Achievements
  - Contact Information
  - End Summary

## Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Vasanthpanuganti/Vasanthpanuganti-Portfolio.git
   cd Vasanthpanuganti-Portfolio
   ```

2. **Add your images**
   - Place your profile picture in `assets/` as `profile.jpg`
   - Add project images as `project1.jpg`, `project2.jpg`, `project3.jpg`
   - See `assets/README.md` for details

3. **Add your resume PDF**
   - Export your resume as PDF
   - Save it as `Vasanth_Panuganti_Resume.pdf` in the `resume/` folder
   - The download button will work automatically
   - See `resume/README.md` for details

4. **Customize content**
   - Edit `index.html` to add your personal information
   - Update career highlights, projects, resume details, and contact info
   - Modify colors in `styles.css` if desired

5. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server for development

## Customization Guide

### 1. Personal Information
Edit the following sections in `index.html`:
- Hero section: Update name, title, and introduction
- Career Highlights: Add your achievements and milestones
- Resume: Fill in work experience, education, and skills
- Projects: Add your project details and links
- Contact: Update email, phone, location, and social links

### 2. Colors and Styling
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;    /* Main brand color */
    --secondary-color: #7c3aed;  /* Accent color */
    --accent-color: #06b6d4;     /* Highlight color */
}
```

### 3. Images
- Add your images to the `assets/` folder
- Update image paths in `index.html` if using different filenames
